# example-scrapers

# Install
 ```
 git clone https://github.com/coderpaddy/example-scrapers
 cd example-scrapers
 pip install requests_html
 ```
 
# Usage
 - Single Scrape
 ```
 python3 scrape_single.py
 ```
 
 - Multiple Scrape
 ```
 python3 scrape_multiple.py
 ```
 
 - Multiple Scrape and Save to CSV
 ```
 python3 scrape_multiple_csv.py
 ```
